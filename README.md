
# CND Uniforms

## Development

### Backend

```bash
cd backend
npm run migrate
npm run seed
npm install
npm start
```

### Frontend

```bash
cd frontend
npm install
npm start
```

## Docker Compose Setup

### To start the containers

```bash
docker-compose up -d
```

### To stop the containers

```bash
docker-compose down
```

Go to `http://localhost:5173` to access the application.
